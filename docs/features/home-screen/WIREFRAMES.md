## Color Reference

**Background Colors:**
- Primary: `#0a0a0a` (zinc-950)
- Card: `#18181b80` (zinc-900/50)
- Empty placeholder: `#18181b4d` (zinc-900/30)
- Hover: `#27272a` (zinc-800)

**Text Colors:**
- Primary: `#f4f4f5` (zinc-100)
- Secondary: `#a1a1aa` (zinc-400)
- Tertiary: `#71717a` (zinc-500)
- Muted: `#52525b` (zinc-600)

**Border Colors:**
- Default: `#27272a` (zinc-800)
- Dashed: `#3f3f46` (zinc-700)
- Hover: `#52525b` (zinc-600)

**Accent Colors:**
- Star: `#eab308` (yellow-500)
- Focus ring: `#52525b` (zinc-600)

---

## Typography Reference

**Font Family:**
- Sans: Geist Sans (default)
- Mono: Geist Mono (code, numbers)

**Text Sizes:**
- Modal title: 20px (text-xl)
- Section header: 20px (text-xl)
- Card title: 18px (text-lg)
- Body text: 16px (text-base)
- Small text: 14px (text-sm)
- Micro text: 12px (text-xs)

**Font Weights:**
- Headings: 600 (font-semibold)
- Emphasis: 500 (font-medium)
- Body: 400 (font-normal)

---

## Spacing Reference

**Container:**
- Mobile padding: 16px (px-4)
- Desktop max-width: 640px (max-w-screen-sm)

**Gaps:**
- Favorite slots: 16px (gap-4)
- Modal grid: 16px (gap-4)
- Section spacing: 64px (space-y-16)

**Border Radius:**
- Small: 4px (rounded)
- Medium: 8px (rounded-lg)
- Large: 12px (rounded-xl)
- Full: 9999px (rounded-full)

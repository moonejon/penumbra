================================================================================
KEY DESIGN ELEMENTS
================================================================================

HERO SECTION
- Profile picture (120px mobile, 160px desktop, rounded-full)
- Owner name (text-2xl, font-bold, centered)
- Bio text (200-300 chars, text-zinc-400, max-w-prose)

FAVORITE BOOKS
- Year filter dropdown ("of 2025" / "of all time")
- 5-6 book covers in responsive layout
  - Mobile: Horizontal scroll
  - Tablet: 4-column grid
  - Desktop: 6-column grid (single row)
- Click opens book details modal

READING LISTS
- Curated reading lists with preview
- Card layout: 3 covers + title + description + count
- Grid layout:
  - Mobile: Single column
  - Tablet/Desktop: 2-column grid

================================================================================
DESIGN SYSTEM
================================================================================

COLORS (Zinc Palette)
- Background: bg-zinc-950
- Text Primary: text-zinc-100
- Text Secondary: text-zinc-400
- Text Tertiary: text-zinc-500
- Borders: border-zinc-800
- Hover Borders: hover:border-zinc-700

TYPOGRAPHY (Geist Fonts)
- Hero name: text-2xl sm:text-3xl font-bold
- Section headers: text-xl font-semibold
- Body: text-base sm:text-lg text-zinc-400
- Small: text-sm, text-xs

SPACING
- Container: max-w-screen-sm mx-auto px-4
- Top padding: pt-20 (clears header)
- Section spacing: space-y-16
- Card padding: p-5

ANIMATIONS (Motion v11)
- Hero: Fade + slide up (spring, 400ms)
- Favorites: Stagger fade + scale (50ms delay/book)
- Lists: Sequential fade + slide (200ms delay)
- Hover: scale(1.05) on covers
- Respects prefers-reduced-motion

================================================================================
RESPONSIVE BEHAVIOR
================================================================================

BREAKPOINTS
- Mobile: < 640px (sm)
- Tablet: 640px - 768px (md)
- Desktop: >= 768px (md) and >= 1024px (lg)

LAYOUT CHANGES
- Hero: 120px -> 160px profile, text-base -> text-lg
- Favorites: Horizontal scroll -> 4-col grid -> 6-col grid
- Lists: Single column -> 2-column grid

================================================================================
DATABASE SCHEMA (NEW)
================================================================================

Reading lists require new tables:

model ReadingList {
  id          String   @id @default(cuid())
  ownerId     Int
  title       String
  description String?
  visibility  ListVisibility @default(PUBLIC)
  createdAt   DateTime @default(now())
  updatedAt   DateTime @updatedAt
  books       ReadingListBook[]
}

model ReadingListBook {
  id            Int         @id @default(autoincrement())
  readingListId String
  bookId        Int
  order         Int         @default(0)
  addedAt       DateTime    @default(now())
  @@unique([readingListId, bookId])
}

enum ListVisibility {
  PRIVATE
  PUBLIC
  UNLISTED
}

================================================================================
ACCESSIBILITY (WCAG 2.1 AA)
================================================================================

REQUIREMENTS
- Color contrast >= 4.5:1 for text
- Keyboard navigation (Tab, Enter, Space, ESC, Arrows)
- Focus indicators visible (ring-2 ring-zinc-600)
- Screen reader support (ARIA labels)
- Touch targets >= 44x44px
- Reduced motion support

KEY ARIA ATTRIBUTES
- aria-label on sections
- aria-haspopup="true" on dropdown
- aria-expanded on dropdown toggle
- alt text on all images
- Semantic HTML (section, h1, h2, article)

## Responsive Transition

### At 1023px (Just Below Breakpoint)
```
MOBILE BEHAVIOR:
- Full-screen overlay
- List hidden when details open
- Backdrop visible
```

### At 1024px (At Breakpoint)
```
DESKTOP BEHAVIOR:
- Side-by-side layout appears
- List stays visible
- No backdrop
- Details panel becomes sticky
```

## Interaction Flow

### Opening Book Details

**Desktop:**
```
1. User clicks book card
2. Card highlights with selected state
3. Details panel appears in right column
4. Details panel sticks to top-24
5. List remains visible and scrollable
6. User can click another book to switch details
```

**Mobile:**
```
1. User clicks book card
2. List view fades out
3. Details overlay fades in with backdrop
4. Details centered on screen
5. User clicks [X] or presses Escape to close
6. Details overlay fades out
7. List view fades back in
```
